=== My gmap ===



License: GPLv2

== Description ==



Créez des cartes Google Maps en enregistrant une latitude et une longitude dans l'administration.



== Installation ==



1/ Téléchargez le dossier de l'extension dans le répertoire wp-content/plugins.

2/ Activez l'extension.

3/ Allez dans l'onglet My gmap.

4/ Créez une carte avec la longitude et la latitude.

5/ Récupérez le shortcode et insérez le sur les pages de votre site.
